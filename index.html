Ваш Session ID (пример)

Ниже показан пример Session ID — это не ваш реальный идентификатор. Он нужен только для наглядности формата.

Формат: длинная строка в шестнадцатеричном (hex) виде, обычно разделённая на удобочитаемые блоки.

Как найти свой реальный Session ID

🔍 Как найти свой реальный Session ID: Откройте приложение Session → нажмите на аватарку (или иконку меню) → ваш ID указан вверху экрана. Скопируйте его.
